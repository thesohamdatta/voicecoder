# Change Log

All notable changes to the VoiceCoder extension will be documented in this file.

## [0.1.0] - 2026-02-13

### Added

- Initial release
- Multi-provider LLM support (Anthropic, OpenAI, Google, Ollama)
- Secure API key management
- Code analysis functionality
- Token usage and cost tracking
- Streaming response support

### Providers

- Anthropic Claude (Sonnet 4.5, Opus 4.5)
- OpenAI (GPT-4 Turbo, GPT-4o, GPT-3.5 Turbo)
- Google Gemini (2.5 Flash, 2.5 Pro)
- Ollama (CodeLlama, DeepSeek Coder, Qwen2.5 Coder)

### Commands

- Select AI Provider
- Ask About Selected Code
- Show Usage & Costs
- Toggle Voice Interface (placeholder)
